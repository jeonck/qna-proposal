# QnA Proposal

## Description
This project contains a Question and Answer system, likely related to the "베트남 온라인 NPL 플랫폼 고도화 사업 감리" (Vietnam Online NPL Platform Advancement Project Supervision). It includes an `index.html` for the frontend, `app.js` for JavaScript logic, `style.css` for styling, and `qna-data.md` for the Q&A content.

## Installation
To set up this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/jeonck/qna-proposal.git
    cd qna-proposal
    ```

2.  **Open `index.html`:**
    Since this is a client-side application, you can simply open the `index.html` file in your web browser.
